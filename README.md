Diabetic Retinopathy (DR) is a diabetes-related eye disease that can lead to vision loss or blindness if not detected early. Manual screening requires expert ophthalmologists and is time-consuming.

This project presents an **AI-powered web application** that uses **Deep Learning and Transfer Learning (Xception Model)** to automatically classify retinal fundus images into different stages of Diabetic Retinopathy.

The system provides an accessible, fast, and automated prediction mechanism through a Flask-based interface.
